## SkyTime Modifications

The STL files distributed with SkyTime Version 1.0 have been substantially modified to support:

- Dual SkyTime installation
- 10-inch rack mounting
- Front panel display visibility
- Front panel recessed button access
- Rear Ethernet access
- Rear GPS antenna access
- Rear USB access
- Waveshare ESP32-P4 ETH PoE mounting retainers
- GPS module retainers
- Integrated top cover

All SkyTime-specific modifications were performed by RPMof78.
